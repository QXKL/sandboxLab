# API 安全代码示例

本目录包含三个演示程序，帮助你理解API安全实践。

## 文件说明

- **JwtAuthDemo.java** - JWT认证完整流程演示
- **RateLimitDemo.java** - 限流算法实现对比
- **SecurityDemo.java** - 常见安全漏洞与防护对比

## 如何运行

### 运行 JwtAuthDemo

```bash
# 编译
javac JwtAuthDemo.java

# 运行
java JwtAuthDemo
```

**演示内容**：
- JWT的生成与验证
- Token过期处理
- Refresh Token机制
- 安全注意事项

### 运行 RateLimitDemo

```bash
# 编译
javac RateLimitDemo.java

# 运行
java RateLimitDemo
```

**演示内容**：
- 固定窗口限流（展示临界点问题）
- 滑动窗口限流（推荐方案）
- 令牌桶限流（支持突发流量）
- 性能对比

### 运行 SecurityDemo

```bash
# 编译
javac SecurityDemo.java

# 运行
java SecurityDemo
```

**演示内容**：
- SQL注入攻击与防护
- XSS攻击与防护
- 权限验证（IDOR防护）
- 密码存储（加盐哈希）
- 输入验证

## 预期输出

每个程序都会对比展示：
- **坏的实现**：存在安全漏洞
- **好的实现**：正确的防护措施
- **攻击演示**：展示如何利用漏洞
- **防护效果**：展示防护后的结果

## 注意事项

1. 这些是**教学演示代码**，为了清晰展示原理，做了简化
2. 生产环境应使用成熟的安全框架（如Spring Security）
3. JWT实现使用了简化的Base64编码，实际应使用标准JWT库
4. 密码哈希演示使用简化算法，实际应使用bcrypt

## 学习建议

1. 先阅读 `doc_01.md`、`doc_02.md`、`doc_03.md`
2. 按顺序运行三个示例程序
3. 阅读代码注释，理解每个安全措施的原理
4. 尝试修改代码，测试不同的攻击场景
5. 思考如何在自己的项目中应用这些安全措施
