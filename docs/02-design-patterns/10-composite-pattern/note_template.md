## 代码示例总结

### 示例1：文件系统

**统一接口**：
```java
interface FileSystemNode {
    void display(String indent);
    long getSize();
}
```

**关键点**：
- File（叶子）：直接返回大小
- Folder（容器）：递归计算所有子节点大小

---

### 示例2：公司组织架构

**统一接口**：
```java
interface OrganizationComponent {
    void printStructure(String indent);
    double getTotalSalary();
}
```

**关键点**：
- Employee（叶子）：返回自己的薪资
- Department（容器）：递归统计所有下属薪资
