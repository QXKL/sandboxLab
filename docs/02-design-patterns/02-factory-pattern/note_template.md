**💡 记住**：
> 简单工厂一生多，  
> 工厂方法一生一，  
> 抽象工厂一生族，  
> 产品配套不分离。

**🎯 核心思想**：将对象创建与使用分离，降低耦合度。

**⚡ 使用原则**：
- 固定不变 → 简单工厂
- 频繁扩展 → 工厂方法
- 配套使用 → 抽象工厂
