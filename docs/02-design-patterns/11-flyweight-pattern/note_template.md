---

### 2. 享元模式的三个角色

1. **Flyweight（抽象享元）**：
2. **ConcreteFlyweight（具体享元）**：
3. **FlyweightFactory（享元工厂）**：

**关键代码**：
```java
class FlyweightFactory {
    Map<String, Flyweight> pool = new HashMap<>();
    
    public Flyweight get(String key) {
        if (!pool.containsKey(key)) {
            pool.put(key, new ConcreteFlyweight(key));
        }
        return pool.get(key);  // 返回共享对象
    }
}
```

---

| **例子** |  |  |  |

**记忆口诀**：
> 单例全局唯一，享元按状态共享，对象池循环用。
